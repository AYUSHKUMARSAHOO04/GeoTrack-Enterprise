# Changelog

All notable changes to GeoTrack Enterprise will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Monorepo structure with frontend (Next.js 15) and backend (FastAPI)
- Frontend design system with dark/light mode, color ramps, and animations
- Backend core configuration with Pydantic Settings
- Async SQLAlchemy 2 database setup with connection pooling
- Alembic migration framework with initial migration
- Docker Compose orchestration for full-stack and dev-only configurations
- GitHub Actions CI/CD pipelines for frontend and backend
- Landing page with hero, features, and architecture sections
- TypeScript type definitions for all domain entities
- Zustand stores for auth and UI state
- TanStack Query provider for server state management
- Comprehensive documentation (README, Architecture, API, Database, Deployment, Contributing)
